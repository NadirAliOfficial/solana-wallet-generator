# Solana Wallet Generator

A simple Python script to generate new Solana wallets (keypairs) — useful for development, testing, and airdrop setups.

## Features
- Generate Solana keypair (public + private key)
- Export in base58 and JSON formats
- Batch generation support

## Requirements
```
pip install solders
```

## Usage
```bash
python generate.py
python generate.py --count 10  # generate 10 wallets
```
